## 🚀 About The Project

> **Rectum** is a front-end clone of the official [Electrum Bitcoin Wallet](https://electrum.org) website — built using pure **HTML**, **CSS**, and **JavaScript**.

Electrum is one of the oldest and most trusted Bitcoin wallets in existence, first launched in **2011**. This project recreates the look and feel of the Electrum website with:

- 🎠 A smooth **auto-sliding image carousel** hero section
- ⚠️ An **animated warning bar** for user awareness
- 🧩 A clean **features grid** showcasing wallet capabilities
- 📄 Separate **Download** and **Documentation** pages
- 📱 **Responsive** layout across devices

## ✨ Features

| Feature | Description |
|---|---|
| 🎠 **Slideshow Hero** | Auto-cycling carousel with 8 slides & overlay text |
| ⚡ **Fast & Lightweight** | No frameworks — pure HTML/CSS/JS |
| 🔒 **Security Banner** | Warning bar alerting users about fake download sources |
| 📦 **Feature Grid** | 9-card grid highlighting wallet capabilities |
| 🌐 **Multi-page** | Home, Download, and Documentation pages |
| 📱 **Responsive** | Adapts cleanly to different screen sizes |

## 🛠️ Getting Started

### Prerequisites

No dependencies needed! Just a browser. 🎉

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Amrit-raj50/rectum.git

# 2. Navigate to the project folder
cd rectum

# 3. Open index.html in your browser
start index.html   # Windows
open index.html    # macOS
xdg-open index.html  # Linux
```

Or just drag `index.html` into your browser! 🖱️
