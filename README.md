⚡ *Electrum Website Clone | HTML & CSS Project*

In this video, I recreated the *Electrum website UI* using **HTML and CSS**. This project focuses on building a clean and responsive layout inspired by the design of the Electrum website.

Through this project you will learn:
• Structuring a real-world website using *HTML*
• Styling modern layouts with *CSS*
• Creating responsive sections
• Designing components like *Navbar, Hero Section, Information Sections, and Footer*

This project is great for beginners who want to *practice frontend development* by cloning real websites and improving their **HTML & CSS skills**.
